# FlaskSentimentVADER
A boilerplate Flask application for Sentiment Analysis using VADER.
Terminal Commands to get started:
> $ cd path/to/FlaskSentimentVADER
> $ python -m venv env
> $ virtualenv env
> $ source env/bin/activate
> $ pip install flask
> $ pip install -r requirements.txt
> $ FLASK_APP=app.py
> $ flask db init
> $ flask db migrate
> $ flask db upgrade
> $ flask run
